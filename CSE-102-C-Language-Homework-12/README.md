# Assignment 12: Comprehensive Data Management System

## Overview
This project is an advanced C programming assignment developed for the CSE 102 Computer Programming course. As a late-term project, it focuses on the practical application of complex data structures and robust memory management techniques. The program involves parsing structured data, dynamically managing records in memory, and ensuring data persistence through advanced File I/O operations.

## Key Features
* **Dynamic Data Allocation:** Replaces static arrays with dynamic memory allocation, allowing the system to handle datasets of arbitrary sizes without hardcoded buffer limitations.
* **Modular Architecture:** The codebase is strictly divided into specialized, single-purpose functions, ensuring high maintainability and adhering to software engineering best practices.
* **Advanced File I/O:** Features secure and efficient reading and writing mechanisms to persistent storage, maintaining database integrity across multiple program executions.
* **Robust Error Handling:** Implements comprehensive input validation and boundary checking to prevent unexpected crashes, segmentation faults, and memory leaks.

## Technical Implementation
This assignment demonstrates proficiency in low-level system operations:
* **Pointers & Memory Management:** Heavily utilizes pointer arithmetic, `malloc()`, `calloc()`, and meticulously planned `free()` routines to optimize system memory usage safely.
* **Struct-Based Organization:** Encapsulates related data fields into custom `struct` definitions, mimicking object-oriented concepts within a procedural C environment.
* **Algorithmic Efficiency:** Incorporates optimized searching, sorting, and traversal algorithms specifically tailored for dynamic data structures.
