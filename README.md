# CSE 102: Computer Programming (C Language) 🚀

Welcome to my portfolio of C programming assignments completed for the **CSE 102 - Computer Programming** course. 

This repository serves as a comprehensive record of my progression from fundamental programming logic to advanced, low-level system design. Over the course of 14 assignments, I have tackled complex computational problems, strictly adhering to academic constraints and software engineering best practices.

## 🧠 Core Concepts & Skills Demonstrated
Throughout these projects, I have developed and applied deep technical knowledge in:
* **Fundamentals:** Control flow, functions, and modular programming logic.
* **Memory Management:** Pointers, array manipulation, and dynamic memory allocation (`malloc`, `calloc`, `free`).
* **Data Structures:** Custom structs, multi-dimensional arrays, Stacks, and Singly Linked Lists.
* **File I/O:** Persistent data storage, data parsing, and multi-stream file management.
* **Algorithm Design:** Recursion, sorting, Euclidean distance calculations, and string manipulation.

## 📂 Repository Structure
Each of the 14 assignments is housed in its own dedicated folder containing:
* The original `C` source code.
* The assignment brief/requirements (PDF).
* A specific `README.md` detailing the project's logic, mathematical models, and technical implementation.

*Feel free to explore the individual assignment folders to see the code and algorithms in action!*
